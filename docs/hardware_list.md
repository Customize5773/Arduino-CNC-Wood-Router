## **Notes**  
- **Cost-Saving Tips**:  
  - Use **smooth rods + LM8UU bearings** instead of linear rails.  
  - Source second-hand stepper motors.  
- **Safety**: Double-check wiring and secure all mechanical connections before powering on.
